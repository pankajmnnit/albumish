# Albumish

Public website and download page for Albumish.

Albumish is the fastest way to shortlist event photos. The desktop app helps users group similar shots, pick the best ones, and export a clean shortlist.

## Website

This repo is intended to be hosted with GitHub Pages from the repository root.

Expected public URL:

```text
https://pankajmnnit.github.io/albumish/
```

## Downloads

The website links to GitHub Release assets:

```text
https://github.com/pankajmnnit/albumish/releases/latest/download/Albumish-0.1.1-arm64.dmg
https://github.com/pankajmnnit/albumish/releases/latest/download/Albumish.Setup.0.1.1.exe
```

## Repository Scope

This public repo should contain only:

- Static website files
- Public release assets
- GitHub Pages configuration

The desktop application source code should live in a separate private repository.
